# 机器人相关学习

本目录用于管理申博机器人方向相关的学习、代码复现、项目 demo 和后续个人网站素材。

目录组织原则：

- 一个项目只保留一个项目文件夹。
- 每个阶段有一个阶段总入口。
- 每个项目目录内统一放 `README.md`、`学习路径.md`、`source/`、`notes/`、`scripts/`、`notebooks/`、`figures/`、`results/`。
- 外部开源代码统一放在对应项目的 `source/` 下。
